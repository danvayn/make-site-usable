WIP to make websites with good content but bad UX considerations usable again.

(Stylus is a browser extension to restyle the websites via CSS themes, compatible with Chrome, Firefox, and any Chromium-based browsers.)

For more documentation, screenshots on the original forked project [you can go here](https://github.com/openstyles/stylus).
